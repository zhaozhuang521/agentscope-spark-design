#### Built-in file types

Supported built-in file types (via `type`):

- `common`
- `epub`
- `excel`
- `html`
- `image`
- `md`
- `mobi`
- `pdf`
- `ppt`
- `txt`
- `web`
- `word`
- `folder`
- `folderGray`
